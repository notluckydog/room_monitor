# room_monitor
Django python 后台开发
#项目用途：
1.作为web后端，提供服务
2.提供二氧化碳、温度、湿度等传感器信息的实时与历史数据查询
3.提供签到功能，实现每日打卡记录等
4.提供记账功能，实现记账数据查询
5.提供完整的用户管理功能
6.微信提醒功能

数据来源：
1.温湿度等环境数据来自于树莓派与传感器搭建的系统
2.天气信息来自于和风天气API提供的数据
3.签到数据来自于每日自己打卡
4.记账功能来自于每日记录
5.短信验证码来源于云片网


技术路径：
1.整体采用Django作为框架，使用Djangorestframework来实现序列化API
2.开发语言使用python
3.使用djangorestframework-jwt来实现token
4.数据库使用sqlite3
5.采用Nginx作为服务器部署

项目构成：
1.myuser：
a.提供登录登出功能
b.提供短信注册等相关功能
c.提供权限管理，用户认证等功能

2.datashow:
a.提供温湿度、二氧化碳浓度实时查询
b.提供天气信息查询
c.提供一段时间内温湿度、二氧化碳浓度等的历史数据查询

3.sign_in:
a.提供每日习惯例如早起等的数据上传功能
b.提供每日身体状况例如心情、身体状态的数据上传功能
c.提供最近一个月习惯数据查询
d.提供最近一个月身体状况数据查询

4.account:
a.提供每日消费情况上传功能
b.提供每日收入情况上传功能
c.提供最近一个月收入与支出数据查询功能
